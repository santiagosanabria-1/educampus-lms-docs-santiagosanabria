> # Guía de Contribución - EduCampus LMS
> 
> Esta guía detalla el flujo de trabajo para colaborar en el proyecto.
> 
> ## Cómo crear una rama de trabajo
> 1. Asegúrese de estar en la rama estable (develop).
> 2. Use nombres descriptivos con prefijos acordados.
> 
> ## Cómo crear un Pull Request
> - Todo cambio debe solicitar revisión mediante un PR detallado hacia develop.
> - Use la plantilla oficial del repositorio.
